# Object Detection Research

To run this, please create a virtual environment and set the ipykernel to the virtual environment.

## Virtual Environment Creation:

### Mac/Linux:
```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

### Windows
```bash
python3 -m venv venv
./venv/Scripts/Activate/ps1
pip3 install -r requirements.txt
```
